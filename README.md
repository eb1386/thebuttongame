# The Button Game

This repository contains **The Button Game**, one of my first attempts at building a simple web game when I was younger.  
It was created as a small experiment to learn basic web development and game logic.  
The project focuses on a simple interactive concept centered around pressing a button and reacting to what happens next.  
Some parts are rough or experimental since it was built while I was still learning.  
This repository exists mainly as an early project and a reminder of where I started with coding.
